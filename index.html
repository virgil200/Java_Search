🔎 Ethical public-records discovery

Search official public sources, not private data.

Enter a name or entity and this Java site prepares direct search paths for court records, business registries, professional licenses, academic-public profiles, sanctions/watchlists, and other public databases. It does not bypass paywalls, hack systems, scrape restricted sites, or decide whether someone has a criminal record.

Privacy-first prototype: no database, no tracking, no external JavaScript, and no sensitive identifiers required.
Important: A matching name can be a false positive. Verify identity with official records, case numbers, dates, licensing numbers, and consent where required.
Not for: harassment, stalking, doxxing, discrimination, circumventing official access controls, or illegal employment/credit/tenant screening.
Use names or entity names only. Do not enter SSNs, passport numbers, private account numbers, passwords, or private addresses.
Public-record access differs by country, state, province, city, and agency.
Backend endpoint: POST /api/search. Source registry: GET /api/sources.
Results will appear here.
This app returns official/public source links and verification steps. It does not store searches or create a permanent dossier.
